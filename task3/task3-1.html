Inline Elements:

  1. Inline elements flow along with the text horizontally, without creating line breaks before or after themselves.
  2. They do not start on a new line.
  3. Examples of inline elements include span, a, img, strong, em, etc.

Block Elements

  1. Block elements always start on a new line and occupy the full width available.
  2. Examples of block elements include div, p, ul, li, h1 to h6, form, etc
  3. Block elements can have width, height, margins, and padding applied to them.
